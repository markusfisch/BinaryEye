Працює в портретній та альбомній орієнтації, може зчитувати інвертовані коди,
має Material Дизайн і також може генерувати штрихкоди.

Використовує ZXing-C++ ("Zebra Crossing") бібліотека сканування штрихкодів.
Підтримувані формати штрихкодів: AZTEC, CODABAR, CODE 39, CODE 93, CODE 128,
DATA MATRIX, EAN 8, EAN 13, ITF, PDF417, QR CODE, RSS 14, RSS EXPANDED,
UPC A, UPC E and UPC EAN EXTENSION.

Це проект із відкритим вихідним кодом:
https://github.com/markusfisch/BinaryEye
