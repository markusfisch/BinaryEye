يعمل في الاتجاهين الرأسي والأفقي، ويمكنه قراءة الرموز المقلوبة، 
ذو تصميم Material ويمكنه أيضًا إنشاء رموز شريطية (باركود).

يستخدم مكتبة ZXing-C++ (المعروفة بـ "Zebra Crossing") لمسح الرموز الشريطية. 
تنسيقات الرموز المدعومة تشمل:  
AZTEC، CODABAR، CODE 39، CODE 93، CODE 128،  
DATA MATRIX، EAN-8، EAN-13، ITF، PDF417، QR CODE،  
RSS 14، RSS EXPANDED، UPC-A، UPC-E، وUPC EAN EXTENSION.
