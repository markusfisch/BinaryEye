可以用縱向和橫向的方式掃描條碼，也能讀取反向碼，
是 Material Design 的風格，也可以製作條碼。

基於使用 ZXing (“斑馬線交叉”) 條碼掃描函式庫。
支援的條碼格式為: AZTEC, CODABAR, CODE 39, CODE 93, CODE 128,
DATA MATRIX, EAN 8, EAN 13, ITF, PDF417, QR CODE, RSS 14, RSS EXPANDED,
UPC A, UPC E 和 UPC EAN EXTENSION.

此項目為開放原始碼:
https://github.com/markusfisch/BinaryEye
